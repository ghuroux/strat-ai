## 1. Overview

### Purpose

The Admin Portal enables organization administrators to manage users, groups, model access, usage tracking, and AI context/system prompts for their organization.

### Target Users

| Role | Access Level | Capabilities |
|------|--------------|--------------|
| Owner | Full | All admin features + billing + delete org |
| Admin | High | User/group management, settings, usage (no billing) |
| Member | None | No admin access |

### Design Principles

1. **Familiar patterns** - Left sidebar nav (industry standard)
2. **Clear hierarchy** - Grouped sections reduce cognitive load
3. **Separation from app** - Distinct layout, clear "back to app" action
4. **Scalable** - Structure accommodates future features
5. **Contextual actions** - Primary actions visible, secondary in menus

### 5.2 Context Assembly Algorithm

```typescript
interface AssembledContext {
  organization?: string;
  groups: Array<{ name: string; prompt: string }>;
  space?: { name: string; context: string };
  area?: { name: string; context: string };
}

async function assembleSystemContext(params: {
  organizationId: string;
  userId: string;
  spaceId?: string;
  areaId?: string;
}): Promise<string> {
  const parts: string[] = [];

  // 1. Organization context (always included)
  const org = await getOrganization(params.organizationId);
  if (org.systemPrompt) {
    parts.push(`[Organization: ${org.name}]\n${org.systemPrompt}`);
  }

  // 2. User's group contexts (all groups they belong to)
  const userGroups = await getUserGroups(params.userId, params.organizationId);
  for (const group of userGroups) {
    if (group.systemPrompt) {
      parts.push(`[Team: ${group.name}]\n${group.systemPrompt}`);
    }
  }

  // 3. Space context (if user is in a space)
  if (params.spaceId) {
    const space = await getSpace(params.spaceId);
    if (space.context) {
      parts.push(`[Space: ${space.name}]\n${space.context}`);
    }
  }

  // 4. Area context (if user is in an area)
  if (params.areaId) {
    const area = await getArea(params.areaId);
    if (area.contextNotes) {
      parts.push(`[Area: ${area.name}]\n${area.contextNotes}`);
    }
  }

  return parts.join('\n\n---\n\n');
}
```
