# Start Development Server

Start the SvelteKit development server for local development.

## Steps

1. Navigate to the stratai-main directory
2. Run `npm run dev` in the background
3. Report the server URL to the user

## Command

```bash
cd /Users/ghuroux/Documents/coding/strathost-ai-main/stratai-main && npm run dev
```

## Notes

- Development server runs on http://localhost:5173
- Hot Module Replacement (HMR) is enabled
- The server will run in the background
